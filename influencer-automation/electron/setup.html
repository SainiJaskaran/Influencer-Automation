One-time setup — connect your database to get started

Quick setup (free):
1. Go to mongodb.com/atlas and create a free account
2. Create a free M0 cluster (any region)
3. Create a database user with a password
4. Click Connect → Drivers → copy the connection string
5. Paste it below

Paste your Atlas connection string. Replace <password> with your actual password. Add /influencerBot before the ? if not present.

This encrypts your login sessions. Use the same secret on all devices to keep the same accounts.
